# Frontend - Teste Desenvolvedor MPlan – To Do (Vue, Vite e Tailwind CSS)

## 1. Tecnologias

- **Frontend:** Vue 3, Vite, Tailwind CSS

## 2. Necessário

- Node.js v16+

## 3. Configuração

1. Dentro da pasta `frontend`, renomeie o arquivo `.env.example` para `.env` e ajuste a variável `VITE_API_URL` conforme a porta do seu backend. Por exemplo:
   ```env
   VITE_API_URL="http://localhost:5000/api"
   ```
2. Rodar o Frontend: Dentro da pasta `frontend` execute os comandos:
   `npm install`
   `npm run dev`

Isso iniciará o frontend.
